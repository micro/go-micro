How Scopes Work

  1. Create a Token with specific scopes (e.g. greeter:read, users:write)
  2. Set endpoint scopes here to require matching scopes for access
  3. Scopes are enforced on all access: API calls (/api/service/endpoint), MCP tools, and the agent
  4. Tokens with * scope bypass all scope checks
  5. Endpoints with no scopes configured are unrestricted
